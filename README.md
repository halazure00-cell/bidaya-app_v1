# Bidaya - Bidayat OS: Sistem Operasi Hati

Selamat datang di Bidaya, sebuah aplikasi pelacak kebiasaan Islami yang dirancang sebagai "Sistem Operasi Manusia" untuk pelacakan spiritual. Aplikasi ini menerjemahkan metodologi kitab klasik Imam Al-Ghazali, "Bidayatul Hidayah" (Permulaan Jalan Hidayah), ke dalam pengalaman pengguna yang modern dan gamified.

## Filosofi

Bidaya bertujuan untuk membantu umat Islam modern dalam perjalanan mereka menuju penyucian diri (tazkiyatun nafs) dengan menyediakan dasbor terstruktur untuk memantau dan merefleksikan amalan harian, adab, dan kondisi spiritual batin.

## Fitur Utama

Aplikasi ini disusun berdasarkan tiga bagian utama dari kitab "Bidayatul Hidayah":

### 1. Boot Sequence & Daily Drivers (Ketaatan)
- **Checklist Interaktif Harian**: Tugas-tugas yang terkunci berdasarkan waktu untuk adab sehari-hari, mulai dari bangun tidur, etiket kamar mandi, wudhu, hingga adab sebelum tidur.
- **Pelacakan Sholat**: Antarmuka sederhana untuk mencatat sholat wajib lima waktu.
- **Log Wirid & Dzikir**: Pencatat digital untuk melacak dzikir harian dengan target yang dapat disesuaikan.

### 2. Hardware & Core Antivirus Scanner (Menjauhi Maksiat)
- **Pemindai 7 Anggota Tubuh**: Fitur refleksi harian untuk memantau penggunaan mata, telinga, lisan, perut, kemaluan, tangan, dan kaki dari perbuatan dosa.
- **Deteksi Malware Inti**: Penilaian mandiri terhadap tiga penyakit hati utama:
    - **Hasad (Dengki)**
    - **Riya' (Pamer)**
    - **'Ujub (Bangga Diri)**

### 3. Network Protocols (Adab Bergaul)
- **Pusat Etiket**: Panduan dan pelacak untuk berinteraksi dengan Allah (vertikal) dan sesama manusia (horizontal), termasuk orang tua, ulama, dan masyarakat umum.

## Dasbor Sistem

- **Baterai Cahaya Amal**: Ikon baterai yang terisi berdasarkan ketaatan yang diselesaikan, yang merepresentasikan "cahaya" amal baik.

## Setup & Menjalankan Proyek

1.  **Install dependensi**:
    ```bash
    npm install
    ```
2.  **Jalankan server pengembangan**:
    ```bash
    npm run dev
    ```
3.  Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
