# Panduan Kontribusi untuk Bidaya

Kami sangat senang Anda tertarik untuk berkontribusi pada proyek Bidaya. Setiap kontribusi, sekecil apapun, sangat kami hargai.

## Cara Berkontribusi

Kami menyambut baik berbagai jenis kontribusi, termasuk namun tidak terbatas pada:

-   **Laporan Bug**: Jika Anda menemukan bug, silakan buat *issue* baru di repositori GitHub kami. Sertakan langkah-langkah untuk mereproduksi bug tersebut.
-   **Permintaan Fitur**: Punya ide untuk fitur baru? Buat *issue* untuk mendiskusikannya.
-   **Pull Request**: Jika Anda ingin mengimplementasikan fitur atau perbaikan bug, silakan buat *pull request*. Pastikan untuk menjelaskan perubahan yang Anda buat.

## Alur Kerja Pull Request

1.  **Fork repositori** dan buat branch baru dari `main`.
2.  **Install dependensi** dengan menjalankan `npm install`.
3.  **Buat perubahan Anda**. Pastikan untuk mengikuti gaya kode yang ada.
4.  **Jalankan linter** untuk memastikan kode Anda bersih dengan `npm run lint`.
5.  **Kirim pull request** Anda ke branch `main` dari repositori utama.

## Standar Kode

-   Gunakan **TypeScript** untuk semua kode baru.
-   Gunakan **React Hooks** dan komponen fungsional.
-   Gunakan **Tailwind CSS** untuk styling.
-   Tulis pesan commit yang jelas dan deskriptif.

Terima kasih telah menjadi bagian dari komunitas Bidaya!
